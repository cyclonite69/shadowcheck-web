# Architecture Assets

This directory is reserved for architecture artifacts that are easier to keep as standalone files than inline Markdown.

Intended use:

- exported architecture PDFs
- diagram images such as PNG or SVG
- companion assets for architecture reviews or NotebookLM-generated summaries

Suggested filenames:

- `ShadowCheck_SIGINT_Forensics.pdf`
- `NotebookLM_Mind_Map.png`

Notes:

- Prefer keeping this directory limited to a small number of intentional assets.
- Large binary files should be added only when you explicitly want them versioned in git.
- If an external source of truth exists, add a short note or link here describing it.
